# EWU Lost & Found

A web app for **EWU students and staff** to report, track, and recover lost items easily.

---

## Features
- User registration & login  
- Add, edit, and view lost/found items  
- Categorize items  
- Upload item images  

---

## Tech Stack
- Python, Django  
- HTML, CSS, Bootstrap  
- SQLite
